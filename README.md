# ember-app-explorer
A CLI tool to explore your Ember.js app folders

## Install
```
npm i -g ember-app-explorer

```

You can also use npx like:
```
npx ember-app-explorer
```

## Usage

```
ember-app-explorer
```

Using current folder.
```
eax 
```

Specifying a path
```
eax ~/Users/user/code/my-ember-app
```
